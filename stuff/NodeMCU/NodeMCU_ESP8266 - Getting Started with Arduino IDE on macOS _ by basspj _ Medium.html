NodeMCU/ESP8266 - Getting Started with Arduino IDE on macOS

basspj
Mar 31, 2019 ¬∑ 2 min read

TLDR; install usb driver (CP210x usb-uart), flash NodeMCU firmware, config Arduino IDE for NodeMCU and start hello-world coding

Install USB Driver

download and install CP210x usb-uart
https://www.silabs.com/products/development-tools/software/usb-to-uart-bridge-vcp-drivers

Flash NodeMCU Firmware

download the latest NodeMCU firmware
https://github.com/nodemcu/nodemcu-firmware/releases

cause the official nodemcu-flasher is not maintained and not supported the macOS, should use 3rd-party flasher instead

So download and install .dmg for nodemcu-pyflasher
https://github.com/marcelstoer/nodemcu-pyflasher/releases

the GUI will look like this
https://github.com/nodemcu/nodemcu-firmware/blob/master/docs/en/getting-started.md#nodemcu-pyflasher

Config Arduino IDE for NodeMCU

follow this instruction
https://github.com/esp8266/Arduino#installing-with-boards-manager

in Arduino IDE,

  • select Board
    Tools > Board > NodeMCU 1.0 (ESP-12E Module) (NodeMCU v. ‚â•1) or
    NodeMCU 0.9 (ESP-12 Module) (NodeMCU v.0.9)
  • select Port
    Tools > Port > /dev/cu.usbserial-xxxx

Start hello-world coding

  • open Arduino IDE
  • write this code on sketch file
void setup() {
  // put your setup code here, to run once:
  Serial.begin(115200);
}void loop() {
  // put your main code here, to run repeatedly:
  Serial.println("hello world");
  delay(2000);
}
  • click Serial Monitor as üîé button on the top right, select 115200 baud
  • click Upload as ‚û°Ô∏è button
  • look for hello world in Serial Monitor
  • Done!
